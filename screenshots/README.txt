# Screenshots Directory

Replace these placeholder files with actual screenshots of your SMS Rate Limiter application:

1. dashboard.png - Main dashboard showing account statistics
2. phone-numbers.png - Phone numbers tab with rate limit indicators
3. rate-limits.png - Rate limits information tab

To capture screenshots:
1. Visit your application at https://sms-rate-limiter.fly.dev/
2. Use your operating system's screenshot tool (e.g., Command+Shift+4 on macOS)
3. Save the screenshots with the names above in this directory
4. Commit and push them to GitHub
